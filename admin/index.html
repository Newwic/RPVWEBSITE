จะแก้ข้อมูลยังไง?

1. ตอนนี้เพิ่ม/แก้ Draft ได้

Admin แสดงสินค้าจากเว็บเดิม 16 รายการ และแก้ข้อมูลชั่วคราวในหน้า Admin ได้

2. ถ้าจะใช้กับเว็บจริงตอนนี้

กด Export data file แล้วนำไฟล์ไปแทนที่ data/rpv-products.js จากนั้นค่อย push ขึ้น GitHub

3. หลังเชื่อม Supabase

ปุ่ม Save/Publish จะบันทึกลง Database ได้โดยไม่ต้องแก้ไฟล์หรือ push ทุกครั้ง

สถานะ Phase 1

  • มีหน้า Admin และ Login แบบ noindex
  • มีเมนูหลังบ้านตามโครงสร้างที่ต้องการ
  • ยังไม่เปิดปุ่มบันทึกจริงจนกว่า Supabase และ RLS พร้อม
  • ไม่มี password, service role key, token หรือ secret ใน frontend
