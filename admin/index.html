จะแก้ข้อมูลยังไง?

1. ตอนนี้ดูข้อมูลได้

Admin แสดงสินค้าจากเว็บเดิม 16 รายการ เพื่อให้ตรวจ UI และข้อมูลก่อน

2. ถ้าจะกดบันทึกจริง

ต้องสร้าง Supabase แล้วรัน schema ในโฟลเดอร์ supabase/

3. หลังเชื่อม Supabase

ปุ่มแก้ไขสินค้า รูปภาพ หน้า Home และข้อมูลติดต่อจะบันทึกลง Database ได้

สถานะ Phase 1

  • มีหน้า Admin และ Login แบบ noindex
  • มีเมนูหลังบ้านตามโครงสร้างที่ต้องการ
  • ยังไม่เปิดปุ่มบันทึกจริงจนกว่า Supabase และ RLS พร้อม
  • ไม่มี password, service role key, token หรือ secret ใน frontend
